Events Concepts

A. Mouse Events


1. Onclick

Onclick is a when we click the one button then function can be executed

2. Double click

On Double click is a when we are Double click the button then function can executed

3. MouseOver

When the mouse is move on button then function can executed

3. MouseOut

When the mouse is out on button then function can executed
